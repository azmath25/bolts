Bolt Management

Actions

Instructions:

  1. Domain: Click cells to mark allowed region
  2. Draw Bolt: Click lattice points (90° turns). Click start to close.
  3. Select Bolt: Click any point on a bolt to select it
  4. Assign Signs: Select a bolt, then assign +/- signs
  5. Drag Edges: Click and drag any edge
  6. Swap Rectangle: Select bolt, click two diagonal corners

Features:

  • Multiple bolts supported
  • Bolts can share vertices
  • Signs only show on selected bolt
  • Drag merges points with sign arithmetic
  • Sign 0 = point vanishes
